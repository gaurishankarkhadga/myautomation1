Project Features: Manual Testing Readiness Report
1. Authentication and Onboarding
Instagram Connect (OAuth): Complete flow for connecting Instagram accounts with local session management.

YouTube Connect (OAuth): Complete flow for authenticating and connecting YouTube channels.

Unified Connect Hub: The home route (/) automatically detects connected platforms and directs users to the Chat Hub or connection prompts.

2. AI Chat Hub
Interactive Interface: A functional chatbot interface powered by the Gemini AI backend.

Context-Aware AI: The AI can manage settings and display previews within the chat.

In-Chat BioLink Editing: Users can edit their link-in-bio via a fullscreen overlay without leaving the chat.

Rich Previews: Support for asset cards and automation previews inside the conversation.

3. Instagram Automation
Post Comment Auto-Reply: Automated replies to comments containing specific keywords.

Comment to DM: Automatically sends a Direct Message to users who comment on posts.

Story Mention Handling: Detects and responds to mentions in Instagram Stories.

Scheduled Replies: A background system manages timed replies for "viral tags."

Automation Dashboard: A visual UI to toggle and configure all active rules.

4. YouTube Automation
Video Comment Auto-Reply: Keyword-based replies for comments on YouTube videos.

Settings Panel: A dedicated interface to manage YouTube-specific triggers.

5. Creator Biolinks
Link-in-Bio Builder: Tools for users to create and customize their own landing pages.

Public Routing: Live links available at /p/:username for followers to access.

Rich Media Support: Support for avatars, product images, and other file uploads.

6. Asset Management
Assets Hub: A central page (/assets) to upload and manage digital files.

Upload Endpoints: Backend support for file uploads and public file serving.

7. Brand Deals
Management Dashboard: A UI to browse and organize brand campaigns.

Application Tracking: System to monitor creator applications for specific deals.

8. Settings and Compliance
Advanced Settings: Unified UI for app-wide preferences.

Legal Pages: Static routes for Privacy Policy, Terms, and Data Deletion (required for platform approval).

Recommended Testing Path
OAuth Connections: Test Instagram and YouTube logins first, as these are core dependencies.

Automation Rules: Create a rule and test it using a live sandbox account.

Biolink Live Check: Build a biolink and verify it loads correctly on the public route.
